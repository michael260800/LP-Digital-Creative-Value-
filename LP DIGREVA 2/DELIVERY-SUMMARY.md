## 📞 SUPPORT & NEXT STEPS

### IF YOU GET STUCK:

**Landing Page Won't Load:**
- Check file path is correct
- Verify no special characters in path
- Convert path slashes: / not \
- Try in different browser

**WhatsApp Link Not Working:**
- Verify number is correct (no spaces/dashes)
- Test link in new tab
- Check format: https://wa.me/[number]
- Number should be: +62[...] or country code

**Ads Getting Rejected:**
- Check copy for negative claims
- Remove "guaranteed" without proof
- Avoid "instant results" claims
- Read Meta's ad policies

**Low CTR (< 4%):**
- Change headline immediately
- Test new image/video
- Refine audience targeting
- Wait 3-7 days, don't panic

**High CPC (> $1):**
- Reduce audience size (be more specific)
- Increase budget (algorithm needs data)
- Test different placement
- Switch to video ads

**No Conversions from Leads:**
- Improve WhatsApp response time
- Ask better qualification questions
- Send portfolio/examples
- Follow up immediately
